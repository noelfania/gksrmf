# 미확인·추가 점검 항목

- **상태**: 미확인

아직 충분히 검증하지 않았거나, 사용자 피드백·릴리스 노트에 남아 있는 항목입니다.

## Undo / Redo (`Ctrl+Z`, `Ctrl+Y`)

- 물리키 조합 전환 후 기본 `EDIT` undo/redo에 맡기는 부분이 있다.
- 조합 중·조합 직후·커밋 직후 undo 체감이 자연스러운지 수동 점검 필요.
- 커스텀 undo 스택을 다시 둘지는 미결정.

## `Ctrl+Backspace` 동작 (회귀 점검)

- 조합 중: key-step 되돌림.
- 비조합: 단어 단위 삭제.
- 구두점 경계(`입력한다? 입`) 삭제 동작은 수정 완료, 실제 사용 중 회귀 여부만 점검.

## 조합 경계 시나리오

- 붙여넣기 직후 바로 한글 입력
- 방향키로 캐럿 이동 후 재입력
- 마우스로 중간 위치 클릭 후 입력
